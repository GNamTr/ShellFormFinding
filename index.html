Instructions:
The force on each node is indicated by a grey arrow. Drag on force vectors to change the applied force at each node. Double click on a force vector to type in a value for it.
Fixed nodes are indicated with a black box. Use the "Add/Remove Fixed Constraint" button and select a node to toggle its fixed state.
By default, the structure is loaded under its own weight, turn off effects due to self weight using the "Apply Self-Weight" toggle or change the density (mass/area measured in kg/m2) of the structure with the "Density" slider.

The dynamic simulation uses a mass-spring-damper system to model interactions between nodes through edges. Damping values may range from 1% to 100% of critical damping for each node-node interaction (adjust this value with the "Damping" slider). Calculations of the dynamic simulation are executed in parallel in a GPU fragment shader for increased performance.

Built by Amanda Ghassaei as a homework assignment for Computational Structural Design and Optimization. Code available on Github.
